#  Superstore Sales Analysis & Prediction

# Project Overview

This project analyzes retail sales data and builds a Linear Regression model to predict sales.

---
# Objectives
* Analyze sales and profit trends
* Identify key factors influencing sales
* Visualize category-wise and time-based performance
* Build a Linear Regression model for sales prediction
* Evaluate model performance using standard metrics
___
# Key Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization (Bar & Line Charts)
* Machine Learning Model (Linear Regression)
* Model Evaluation (MAE, MSE, R² Score)

#  Visualizations
* Bar Chart - Sales by Category
* Line Chart - Sales Trend Over Time
* Heatmap - correlation
* Scatter - discount vs profit analysis

# Machine Learning Model
* Algorithm: Linear Regression
Steps:
* Feature selection
* Train-test split
* Model training using Scikit-learn
* Predictions on test data
