#  Superstore Sales Analysis & Prediction

# Project Overview

This project analyzes retail sales data and builds a Linear Regression model to predict sales.

---

# Key Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization (Bar & Line Charts)
* Machine Learning Model (Linear Regression)
* Model Evaluation (MAE, MSE, R² Score)

---

# Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

---

# Results

* Identified sales trends across categories
* Analyzed profit distribution
* Built a predictive model for sales
